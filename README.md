# Amazon Sales Dashboard (2023)

# Project Overview
This project analyzes Amazon sales data to understand sales performance across regions,
product categories, and delivery status, and to support data-driven business decisions.

## Business Problem:
The business wants to track total sales, identify top-selling products and regions,
monitor delivery performance, and analyze order cancellations.

## Dataset Name:
MINI PROJECT DATASET - WEEK-03.xlsx

## Tools Used: Excel
Excel: Data cleaning , transformation and dashboard visualization

## Project Workflow:
1. Understood business requirements
2. Collected and reviewed raw sales data
3. Cleaned data and fixed data types in Excel
4. Transformed data and created calculated columns
5. Built interactive dashboard in Power BI
6. Analyzed insights and prepared findings

## Key KPIs:
Total sales,Total selling products,sales by regions,Avg delivery time, cancellation percentage

## Dashboard Preview:
Amazon sales Dashboard.png

## Key Insights: 
North region generated the highest sales
Smartphones are the top-selling products
South region has faster average delivery times
West region shows a higher cancellation percentage

## Conclusion:
This dashboard helps stakeholders monitor sales performance,improve delivery efficiency,and make informed business decisions using data.
